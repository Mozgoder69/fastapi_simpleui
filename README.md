# Summary

This repository contains a prototype of the “Cleaners” information system,
designed and developed as part of a student's coursework on databases

## Used frameworks and technologies:
### Domain
PostgreSql - Database
AsyncPg - Interface

### API
Uvicorn - Server (ASGI)
FastApi - Routes
Pydantic - Models

### UI
Jinja2 - Templates
AlpineJs - Reactivity
InviseCss - Styling
